# Profile Experience Framework

## Purpose

This folder contains the canonical shared product framework for every Profile experience in Grow. Person, Source, Breeder, and future Entity Profiles use this framework to remain recognizably part of the same product while presenting content appropriate to what they represent.

## Scope

The framework governs:

- Profile composition
- section rhythm
- module behavior
- responsive layout
- visual hierarchy
- dark glass styling
- brand color use
- empty states
- featured-section behavior
- Profile consistency across entity types

## Boundaries

The framework does not:

- redefine canonical identity
- redefine People or Entities
- own Sessions, Vault, Network, Community, or Learn data
- define entity-specific content
- define database implementation
- prescribe application code

Those responsibilities remain with the Grow Foundation, owning product areas, entity-specific specifications, and implementation tasks.
