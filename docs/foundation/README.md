# Grow Foundation

This directory contains the durable philosophy, architectural history, and validated future concepts of the Grow platform.

## Document categories

- **Grow Foundation** — what Grow believes and why it exists
- **Platform Milestones** — durable architectural foundations already completed
- **Foundation Notes** — validated ideas intentionally deferred until the correct phase
- **Individual Foundation Notes** — focused specifications that may later become development work

A Foundation Note is not a casual idea or ordinary backlog item. It records a concept that aligns with the Grow Foundation, creates meaningful platform value, and should not be lost even though it is not being built yet.
